Keamanan

Password saja tidak cukup untuk keamanan anda? Tenang, berikut 4 pilihan fitur keamanan yang kami tawarkan.
Mudah dalam pengaturan dan bisa dipilih aktif pada jenis layanan apa saja. Karena keamanan member kami adalah yang utama.

Master Code

Master code adalah kode induk berupa 6 digit angka yang berfungsi untuk merubah data. Master Code ini unik dan dibuat oleh member.

Pin Login

Pin Login adalah fitur keamanan yang disetting saat anda login. Bisa dikirim via email ataupun SMS atau keduanya.

Pin Transaksi

Pin Transaksi adalah pin yang disetting untuk keamanan transaksi. Bisa dikirim via email ataupun SMS atau keduanya.

API security(Khusus Merchant)

API security memungkinkan anda untuk membuat whitelist atau blocklist terhadap IP Address yang boleh mengakses API anda.
